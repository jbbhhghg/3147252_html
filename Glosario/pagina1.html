    • Términos Informáticos (A-M)

A

Algorithm

Conjunto ordenado de operaciones sistemáticas que permite hacer un cálculo y hallar la solución de un tipo de problemas.

Antivirus

Software diseñado para detectar y eliminar programas maliciosos del sistema.

B

Backup

Copia de seguridad de datos para prevenir pérdidas.

Browser

Programa que permite acceder y visualizar páginas web.

C

Cache

Memoria de acceso rápido que almacena datos frecuentemente usados.

Cookie

Archivo que almacena información sobre la actividad del usuario en un sitio web.

D

Desktop

Interfaz principal del sistema operativo donde se muestran iconos y ventanas.

Download

Proceso de transferir datos desde un servidor remoto a un dispositivo local.

E

Email

Sistema de intercambio de mensajes electrónicos a través de Internet.

Encryption

Proceso de codificación de información para protegerla de accesos no autorizados.

F

Firewall

Sistema de seguridad que controla el tráfico de red.

Framework

Estructura conceptual y tecnológica que sirve de base para el desarrollo de software.

G

Gateway

Puerta de enlace entre diferentes redes.

Graphics

Elementos visuales generados o procesados por computadora.

H

Hardware

Componentes físicos de un sistema informático.

Hosting

Servicio de alojamiento para sitios web y otros contenidos en Internet.

I

Input

Datos introducidos en un sistema informático para su procesamiento.

Interface

Medio de comunicación entre el usuario y el sistema informático.

J

Java

Lenguaje de programación orientado a objetos multiplataforma.

JavaScript

Lenguaje de programación interpretado usado principalmente en desarrollo web.

K

Keyboard

Dispositivo de entrada que permite introducir texto y comandos.

Kernel

Núcleo del sistema operativo que gestiona los recursos del hardware.

L

Laptop

Computadora portátil diseñada para su uso móvil.

Link

Conexión o enlace entre diferentes elementos o páginas web.

M

Memory

Dispositivo que permite el almacenamiento de datos y programas.

Monitor

Dispositivo de salida que muestra información visual al usuario.
