Términos Informáticos (N-Z)

N

Network

Sistema de computadoras y dispositivos interconectados para compartir recursos.

Node

Punto de conexión o intersección en una red informática.

O

Operating System

Software que gestiona los recursos del hardware y proporciona servicios a los programas de aplicación.

Output

Resultado o datos producidos por un sistema informático.

P

Password

Contraseña que protege el acceso a un sistema o cuenta.

Protocol

Conjunto de reglas que permiten la comunicación entre dispositivos en una red.

Q

Query

Consulta o petición de información a una base de datos.

Queue

Estructura de datos que organiza elementos en orden de llegada.

R

RAM

Memoria de acceso aleatorio que almacena datos temporalmente.

Router

Dispositivo que dirige el tráfico de datos entre diferentes redes.

S

Software

Conjunto de programas y rutinas que permiten realizar tareas específicas.

Server

Computadora que proporciona servicios a otros dispositivos en una red.

T

Terminal

Interfaz de línea de comandos para interactuar con el sistema.

Thread

Secuencia de instrucciones que puede ser ejecutada en paralelo.

U

URL

Localizador uniforme de recursos que especifica la ubicación de un recurso en Internet.

USB

Estándar de conexión universal para dispositivos periféricos.

V

Virtual Memory

Técnica que utiliza el disco duro como extensión de la memoria RAM.

Virus

Programa malicioso que puede dañar o alterar el funcionamiento del sistema.

W

Web

Sistema de documentos interconectados accesibles por Internet.

Wireless

Tecnología que permite la comunicación sin cables.

X

XML

Lenguaje de marcado extensible para almacenar y transportar datos.

XSS

Tipo de vulnerabilidad de seguridad en aplicaciones web.

X

YAML

Formato de serialización de datos legible por humanos.

YouTube

Plataforma de compartición y visualización de videos en línea.

Z

ZIP

Formato de archivo comprimido.

Zone

Área específica de una red o sistema con características particulares.
